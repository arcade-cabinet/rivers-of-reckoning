Quickstart Guide
================

Welcome to **Rivers of Reckoning**! This guide will help you get into the game 
as quickly as possible.

1. Install Dependencies
-----------------------

First, make sure you have the necessary tools installed (Node.js 18+ and pnpm):

.. code-block:: bash

   pnpm install

2. Run the Game
---------------

From the root of the repository, run:

.. code-block:: bash

   pnpm dev

3. Start Your Adventure
-----------------------

On the title screen, click the **START GAME** button to generate a new procedural 
world and begin your journey.

4. Basic Controls
-----------------

* **WASD Keys**: Move your character.
* **Space**: Attack.
* **ESC**: Pause the game.
